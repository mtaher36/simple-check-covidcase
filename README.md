# Simple Check Covid Statistics

## Features

- Enter the country name in the input field.
- Click the "Covid Statistics" button to fetch and display the covid statistics information.
- Displayed covid statistics information includes active cases, critical cases, new cases, and recovered.

## Tech

- [HTML] - The standard markup language for creating web pages and web applications.
- [Vanilla CSS] - Plain CSS without the use of any preprocessor or framework, providing basic styling capabilities.

- [AJAX for API requests](https://api.jquery.com/Jquery.ajax/) - Asynchronous JavaScript and XML (AJAX) technique used to make requests to the server without reloading the entire page, often used for fetching data from APIs in web development.

## API Used
This project utilizes the [rapidapi.com](https://rapidapi.com/) API to fetch covid-19 statistics based on the provided country name.


## How to Use

1. clone the repository to your local machine:
```sh
https://github.com/mtaher36/simple-check-covidcase.git
```
2. Open the index.html file in your web browser.
3. Enter the name of the country you want to check the covid statistics for in the input field.
4. Click the "Covid Statistics" button.
5. The weather information for the specified country will be displayed.
